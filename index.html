Give it an outcome. It detects your project, assembles the right crew, and shows its reasoning before it acts.

How it works
GitHub
🔍

Glass-box by design

Every routing decision is inspectable. Which role resolved to which provider, on which model, and why. The crew manifest shows the plan before any work begins.

🧩

Multi-runtime

Composes the tools you already have (plugins, agents, MCP servers) and falls back to its own built-ins. Runs on bare Claude Code and gets better as you install more.

🌐

Multi-domain

Not just code. Product, business, content, and operations pipelines are first-class. PRDs, roadmaps, launch plans, runbooks, and more.

⚙️

Code over model

A deterministic Node CLI owns routing, scoring, and validation. No LLM calls. Reproducible by construction. The model is reserved for judgment.

🚦

Four modes

Run plans and shows. Autopilot runs the full lifecycle hands-off. Diagnose fixes one bug failure-first. Audit sweeps the whole codebase.

🔒

No extra services

No separate model API, no runtime to deploy, no keys to manage. It draws your interactive Claude Code subscription. If you can run Claude Code, you can run Muster.
